flex

flex帮助我们设置剩余空间如何进行分配(这里要注意是剩余空间，剩余空间等于父容器的宽度减轻子元素的宽度之和)，水平方向的分配。

flex-grow:用来指定扩展比率，即剩余空间为正值时该子元素所分配到的空间比例。
